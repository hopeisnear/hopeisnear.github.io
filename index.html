This page includes example usages of angular select 2 directive.

No parameters and empty option

Selected value in model: {{country || 'undefined'}}
Markup
<select data-jq-select2="" data-ng-model="country" placeholder="Select country" data-ng-options="country.code as country.name for country in countries"> <option></option> </select>
Data in format:
$scope.countries = [ {name: 'Afghanistan', code: 'AF'}, {name: 'Albania', code: 'AL'}, ... {name: 'Zimbabwe', code: 'ZW'} ];

Preselected value and no empty option available

Selected value in model: {{preselectedCountry || 'undefined'}}
Markup
<select data-jq-select2="" data-ng-model="preselectedCountry" placeholder="Select country" data-ng-options="country.code as country.name for country in countries"> </select>
Data in format:
$scope.preselectedCountry = $scope.countries[5].code;

Dynamic loaded data from json with key-value pair as select 2 options

Use load states button to load data from json file.
States loaded
Selected value in model: {{state || 'undefined'}}
Markup
<select data-jq-select2="statesSelect2Options" data-ng-model="state" data-ng-options="key as value for (key, value) in states" placeholder="Select state"> <option></option> </select>
Data in format:
$scope.states = { "AL": "Alabama", "AK": "Alaska", "AS": "American Samoa", "AZ": "Arizona", ... "WY": "Wyoming" } $scope.statesSelect2Options = { minimumResultsForSearch: -1 };

Disabled/enabled

Use button to disable enable select 2 dropdown

<select class="jq-select2" data-jq-select2="" placeholder="Select..." data-ng-model="someModel" data-ng-disabled="selectDisabled"> <option></option> </select>

Dynamic placeholder

Type in the new placeholder.
Markup
<select class="jq-select2" data-jq-select2="" placeholder="{ {someDynamicPlaceholder} }" data-ng-model="someModel"> <option></option> </select> <input type="text" data-ng-model="someDynamicPlaceholder" placeholder="Type in new select 2 placeholder">
